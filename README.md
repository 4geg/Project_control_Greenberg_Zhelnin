## Описание проекта

Информационная система управления проектами компании предназначена для автоматизации процессов создания проектов, постановки задач, назначения исполнителей, контроля сроков и отслеживания статусов выполнения работ.

Система реализована в формате локального веб-приложения. Пользователь взаимодействует с системой через браузер, а данные хранятся в локальной базе данных SQLite.

## Основной функционал

Система позволяет выполнять следующие действия:

* авторизоваться в системе;
* просматривать список проектов;
* создавать новые проекты;
* редактировать проекты;
* удалять проекты;
* открывать карточку выбранного проекта;
* создавать задачи внутри проекта;
* редактировать задачи;
* удалять задачи;
* назначать исполнителя задачи;
* указывать срок выполнения задачи;
* изменять статус задачи;
* фильтровать задачи по статусу.

## Статусы задач

В системе используются следующие статусы задач:

* Новая;
* В работе;
* На проверке;
* Завершена.

## Быстрый запуск на Windows

Для быстрого запуска проекта необходимо:

1. Скачать или клонировать проект.
2. Открыть папку проекта.
3. Запустить файл:

```text
start.bat
```

После запуска файл автоматически выполнит следующие действия:

* создаст виртуальное окружение `.venv`, если оно отсутствует;
* установит зависимости из `requirements.txt`;
* выполнит миграции базы данных;
* создаст тестового администратора, если он ещё не создан;
* запустит локальный сервер Django;
* откроет приложение в браузере.

После запуска приложение будет доступно по адресу:

```text
http://127.0.0.1:8000/
```

## Ручной запуск проекта

Если автоматический запуск через `start.bat` не используется, проект можно запустить вручную.

1. Перейти в папку проекта:

```bash
cd Project_control_Greenberg_Zhelnin
```

2. Создать виртуальное окружение:

```bash
python -m venv .venv
```

3. Активировать виртуальное окружение:

Для Windows PowerShell:

```bash
.\.venv\Scripts\activate
```

Для Windows Command Prompt:

```cmd
.venv\Scripts\activate.bat
```

4. Установить зависимости:

```bash
pip install -r requirements.txt
```

5. Выполнить миграции базы данных:

```bash
python manage.py migrate
```

6. Создать администратора вручную, если он не был создан автоматически:

```bash
python manage.py createsuperuser
```

7. Запустить сервер:

```bash
python manage.py runserver
```

8. Открыть приложение в браузере:

```text
http://127.0.0.1:8000/
```
